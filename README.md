# Ankh of Eternity（学習用）

Hello Mentor の教材をもとに、Web 制作の基礎力向上を目的として制作しました。

## 概要

このリポジトリは学習用として作成したサイトです。

## 使用技術

- HTML5
- SCSS
- CSS3
- JavaScript

## 工夫した点

- JavaScript によるアニメーションを取り入れ、ユーザー体験を損なわないよう過度にならない動きを意識して実装しました。
- PC・スマートフォンなど、あらゆる画面サイズでレイアウトが崩れないようレスポンシブ対応を行いました。
- デザインカンプの意図を汲み取り、余白や文字組など細部まで忠実な再現を心がけました。
- 保守性を意識し、後から見返した際にも分かりやすいクラス設計・命名を行いました。
